# Coding-Dev-Tools.github.io

GitHub Pages site for **Coding-Dev-Tools** — open-source developer CLI tools.

Static HTML, no build step: each page is a standalone `.html` file; `index.html` is the homepage with tool cards. Guides double as SEO landing pages for the Coding-Dev-Tools tools (apighost, apiauth, click-to-mcp, configdrift, datamorph, deadcode, schemaforge, and friends).

## Editing

Edit the HTML directly and push to `main` — GitHub Pages serves it as-is. Keep new guides consistent: add a homepage card in `index.html` when adding a page.
