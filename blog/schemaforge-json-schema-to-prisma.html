Convert JSON Schema to Prisma without rewriting types by hand

JSON Schema is great for validation. Your database layer speaks Prisma. Bridging the two by hand means guessing type mappings, duplicating enums, and hoping you did not miss an index.

SchemaForge converts JSON Schema definitions into Prisma schema files directly.

Install

pip install git+https://github.com/Coding-Dev-Tools/schemaforge.git
schemaforge convert --from json_schema --to prisma --input schema.json --output schema.prisma

What it does

  • converts JSON Schema to Prisma schema in one command
  • maps JSON Schema types to Prisma types automatically
  • preserves enums, indexes, and unique constraints
  • also converts Prisma back to JSON Schema
  • honest about limits: foreign keys and relations are not yet preserved
  • Python 3.10+, MIT license, 310 passing tests

Why it works

Every format converts to and from a shared internal representation. That makes roundtripping safe and means adding a new format does not require pairwise converters. You add one parser and one generator.
