DevForge: open-source CLI tools that just work

DevForge is the Coding Dev Tools suite of developer command-line tools — config drift detection, API mocking, schema translation, env-secret management, infrastructure diffing, dead-code detection, and more. MIT licensed, Python 3.10+.

Last updated: 2026-07-17 · All tools on GitHub

Not on public PyPI. Each tool installs via a direct GitHub install (or the self-hosted index once published). See each tool's repo for its exact command.

Install the suite

pip install "git+https://github.com/Coding-Dev-Tools/devforge-cli.git[all]"

One devforge command gives you the full CLI toolkit. Individual tools install from their own repos too.
